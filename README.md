# 🎬 Course Video Tracker

Production Pipeline Dashboard for tracking course video creation.

## Local Setup

```bash
# 1. Open terminal in this folder
cd course-video-tracker

# 2. Install dependencies
npm install

# 3. Run the server
npm start

# 4. Open browser
# Go to http://localhost:3000
```

## Deploy to Vercel

Already configured with `vercel.json`. Just push to GitHub and connect to Vercel.
